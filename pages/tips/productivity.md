## CLAUDE.md 활용하기

프로젝트 루트에 `CLAUDE.md`를 작성하면 매 대화에서 반복 설명할 필요가 없습니다.

포함하면 좋은 내용:
- 프로젝트 빌드/테스트 명령어
- 코드 스타일 규칙
- 디렉토리 구조 설명
- 자주 사용하는 패턴

## 컨텍스트 관리

- `/compact`로 긴 대화를 요약하여 컨텍스트 절약
- 새로운 작업은 새 대화에서 시작
- 큰 파일은 필요한 부분만 참조하도록 안내

## Agent 활용

- 복잡한 검색은 Explore 에이전트에 위임
- 독립적인 작업은 병렬 에이전트로 처리
- Plan 모드로 큰 작업의 구현 전략 먼저 수립

## 효율적인 프롬프트

- 구체적으로 요청: "이 함수를 리팩터링해줘" → "이 함수에서 DB 호출을 분리해줘"
- 파일 경로를 명시하면 검색 시간 절약
- 원하는 결과의 예시를 함께 제공
