## 변경사항

- `/usage`에서 이제 limit 사용량을 유도하는 각 카테고리별 분석(skill, subagent, plugin, MCP server별 비용)을 표시합니다
- `/diff` 상세 보기를 이제 키보드로 스크롤할 수 있습니다(화살표, `j`/`k`, `PgUp`/`PgDn`, `Space`, `Home`/`End`)
- Markdown 출력이 이제 GFM 작업 목록 체크박스(`- [ ] todo` / `- [x] done`)를 plain bullets 대신 렌더링합니다
- Enterprise: `managed-mcp.json`과 함께 claude.ai cloud MCP connector를 로드하기 위해 `allowAllClaudeAiMcps` managed setting을 추가했습니다
- PowerShell permission bypass를 수정했습니다: built-in `cd` 함수(`cd..`, `cd\`, `cd~`, `X:`)가 작업 디렉토리를 감지되지 않게 변경하여 이후 명령이 workspace 외부를 읽을 수 있는 문제를 해결했습니다
- git worktree의 sandbox write allowlist가 shared `.git` 디렉토리(`hooks/` 및 `config` 제외)만이 아니라 전체 main repository root을 포함하는 문제를 수정했습니다
- PowerShell prefix/wildcard allow rule(예: `PowerShell(dotnet.exe build *)`)이 native executable과 script을 사전 승인하지 않는 문제를 수정했습니다
- parser가 `cd`/`pushd`/`popd` 전반에서 `PWD`/`OLDPWD`/`DIRSTACK`에 대한 stale variable-tracking 값을 신뢰하는 permission-analysis gap을 수정했습니다
- Bash tool의 `find`가 macOS system file/vnode table을 소진하여 큰 디렉토리 트리에서 host가 crash되는 문제를 수정했습니다
- managed-settings approval dialog가 시작 시 수락 후 terminal이 frozen되는 문제를 수정했습니다
- working tree에 실제 변경사항이 없을 때 `/ultraplan`과 remote session 생성이 "Could not capture uncommitted changes"로 실패하는 문제를 수정했습니다
- script path에 공백이 포함된 경우 `otelHeadersHelper`가 조용히 실패하는 문제를 수정했습니다. helper 실패는 이제 `/doctor`과 debug log에서 보고됩니다
- thinking spinner가 tool call 전반과 새로운 thinking burst로 이어지면서 amber 상태로 유지되는 문제를 수정했습니다
- 많은 짧은 라인이 포함된 출력에서 collapsed Bash output이 잘못된 hidden-line count를 보고하는 문제를 수정했습니다
- slash-command argument-hint가 hint가 input box를 overflow할 때 입력된 문자를 clip하는 문제를 수정했습니다
- argument-hint와 progressive arg suggestion이 frontmatter `name:`이 디렉토리 basename과 다른 skill을 Tab-complete한 후 나타나지 않는 문제를 수정했습니다
- status bar가 skill/agent `effort:` frontmatter에서 적용된 effort level 대신 사용자의 baseline `/effort` setting을 표시하는 문제를 수정했습니다
- Ctrl+O transcript view가 열리는 순간 freeze되는 대신 새로운 메시지를 tail하지 않는 문제를 수정했습니다
- recalled prompt-history entry 편집 시 화살표 키로 더 위/아래로 이동하면 편집이 손실되는 문제를 수정했습니다
- `/config` exit summary가 관련 없는 설정을 toggle할 때 auto-compact 및 theme에 대한 phantom change를 보고하는 문제를 수정했습니다
- `/insights`가 cached session-meta file에서 optional field가 없을 때 crash되는 문제를 수정했습니다
- input이 없는 malformed PowerShell 및 History tool call이 transcript collapsing에서 read로 잘못 분류되는 문제를 수정했습니다
- claude.ai 또는 Claude mobile app에서 Remote Control session 이름을 변경해도 `claude --resume`의 local session name이 업데이트되지 않는 문제를 수정했습니다
- 방금 제출된 prompt가 up-arrow history에 두 번 나타날 수 있는 race condition을 수정했습니다
- fullscreen mode에서 "Jump to bottom" pill을 탭해도 즉시 dismiss되지 않는 문제를 수정했습니다
- `/feedback` report를 개선하여 context compaction 전에 발생한 대화를 포함하여 긴 session에서의 이슈를 더 쉽게 분류할 수 있도록 했습니다
