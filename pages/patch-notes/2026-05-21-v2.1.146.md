## 변경사항

- `/simplify`을 `/code-review`로 이름 변경하였으며, 선택적 노력 수준을 지원합니다 (예: `/code-review high`)
- Auto mode가 더 이상 사용자나 skill이 명시적으로 의존할 때 `AskUserQuestion`을 억제하지 않습니다
- Windows PowerShell tool이 `pwsh`를 winget 또는 Microsoft Store를 통해 설치했을 때 "command line is invalid" 오류로 실패하는 문제를 수정했습니다 (v2.1.124에서의 regression)
- MCP `resources/list`, `resources/templates/list`, 및 `prompts/list`가 페이징 서버에서 page 1 이후의 항목을 손실하는 문제를 수정했습니다
- Windows Terminal의 attached background session에서 Claude가 streaming할 때 전체 화면 깜빡임 문제를 수정했습니다
- auto-updater 상태 표시줄이 업데이트 실패 시 현재 버전을 표시하지 않는 문제를 수정했습니다
- Windows에서 background-job worktree를 제거할 때 NTFS junction을 따라 main repo로 이동하는 문제를 수정했습니다
- `/background`가 유일한 입력이 skill 또는 custom slash command인 session을 거부하는 문제를 수정했습니다
- Backgrounded session이 "don't ask again"으로 이미 부여한 tool 권한을 다시 묻는 문제를 수정했습니다
- `/theme` color editor 및 "New custom theme" dialog이 Esc에 응답하지 않는 문제를 수정했습니다
- Agent SDK를 통해 실행할 때 streaming session 종료 시 발생하는 uncaught exception 문제를 수정했습니다
- `forceLoginOrgUUID` 및 `forceLoginMethod` managed-settings policy가 third-party-provider 및 API-key session에 대해 적용되지 않는 문제를 수정했습니다
- GNOME Terminal의 right-click 및 middle-click paste가 텍스트를 삽입하지 않는 문제를 수정했습니다
- Multi-agent session에서 `CLAUDE_CODE_SUBAGENT_MODEL`이 자식 프로세스로 전달되지 않는 문제를 수정했습니다
- Auto-updater 안정성을 개선했습니다: native version check 및 download가 즉시 실패하는 대신 일시적인 네트워크 오류를 재시도합니다
- 큰 파일 편집에 대한 diff 렌더링 성능을 개선했습니다
