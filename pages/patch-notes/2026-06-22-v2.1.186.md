> 원문: [Claude Code v2.1.186 Release Notes](https://github.com/anthropics/claude-code/releases/tag/v2.1.186)

## 변경사항

- `claude mcp login <name>` 및 `claude mcp logout <name>`을 추가하여 대화형 `/mcp` 메뉴를 열지 않고 CLI에서 MCP 서버를 인증할 수 있도록 지원하며, SSH를 통한 완료를 위해 `--no-browser` stdin redirect 지원이 포함됨
- `/workflows` agent 상세 보기에 상태 필터링(press `f`) 추가
- `/plugin` Installed 탭에 "Skills" 섹션 추가
- `it2` CLI를 찾을 수 없을 때 경고와 함께 `teammateMode: "iterm2"` 설정 추가
- `awsAuthRefresh`가 구성된 경우 `/login`에 "Claude Platform on AWS - refresh credentials" 옵션 추가
- `!` bash 명령어가 이제 Claude가 출력에 자동으로 응답하도록 트리거함; 이전의 context 전용 동작을 유지하려면 settings.json에서 `"respondToBashCommands": false`를 설정
- 머신이 절전 모드에서 깨어난 후 스트리밍 요청이 "Content block not found" 또는 JSON 구문 분석 오류로 실패하는 문제 수정
- subagent 트랜스크립트 스크롤 위치가 종료 시 메인 트랜스크립트로 침범하는 문제 수정
- agent의 계획이 로드되기 전에 배경 작업 미리보기가 원시 도구 이름을 표시하던 문제 수정
- 내부 권한 게이트가 동시 CLI 세션에 대해 꺼져 있을 때 Chrome 탭 그룹 격리가 적용되지 않는 문제 수정
- 배경 세션 요약이 중복되는 문제 수정; agent의 자체 턴 종료 요약이 이제 요약 라인으로 표시됨
- `claude agents`에서 배경 세션을 열 때 이전 화면이 남아 있는 문제 수정
- `Agent(type)` deny 규칙 및 `Agent(x,y)` allowed-types 제한이 명명된 subagent spawn에 적용되지 않는 문제 수정
- 메인 턴이 끝난 후 배경 agent가 여전히 실행 중일 때 Esc 및 Ctrl+C가 응답하지 않는 문제 수정
- 권한 프롬프트에서 옵션 텍스트가 overflow할 때 옵션 번호가 정렬되지 않는 문제 수정
- agent 패널에서 완료된 subagent에 `x`를 눌러도 해제되지 않는 문제 수정
- 이전 세션을 재개할 때 의도적으로 폐기된 도구에 대한 오해의 소지가 있는 "MCP server disconnected" 알림 수정
- `/plugin` Installed에서 이미 최상단으로 스크롤된 상태일 때 "more above" 표시기가 나타나는 문제 수정
- `~~strikethrough~~`가 assistant 메시지에서 리터럴 틸드를 보여주는 대신 strikethrough로 렌더링되도록 수정
- `--tools`가 cold first launch에서 플래그가 로드되기 전에 feature-gated 도구를 통과시키는 문제 수정
- `claude agents`의 배경 작업 상태가 회신 후 stale "needs input" 메시지를 표시하는 문제 수정
- light terminal에서 `claude agents`의 배경 세션을 열 때 dark-theme flash가 나타나는 문제 수정
- `claude agents`에서 마우스로 선택한 텍스트를 삭제한 후에도 강조 표시가 유지되는 문제 수정
- usage-based Enterprise 및 Team 구독자에 대해 세션 비용이 표시되지 않는 문제 수정
- agent teams 수정: tmux/pane backend를 통해 spawn된 teammates는 이제 leader의 `--effort` 레벨을 상속함
- Workflow `agent({schema})` subagent가 반복된 schema 유효성 검사 실패 시 5회 시도 후 중단하지 않고 무한 루프에 빠지는 문제 수정
- `claude mcp get` 및 `claude mcp remove`를 개선하여 오타 시 가장 가까운 구성된 서버 이름을 제안하고 긴 서버 목록을 자름
- 메모리 개선: 크기 제한에 가까워질 때 agent가 `MEMORY.md` 인덱스를 압축하도록 상기받음
- skill frontmatter 개선: `display-name`, `default-enabled`, `fallback`, 및 `metadata.*` 키가 이제 kebab-case, snake_case, camelCase를 허용함
- 형식이 잘못된 `SKILL.md` YAML frontmatter 처리 개선: 자동 실패하지 않고 빈 메타데이터로 skill 본문을 로드함
- `CLAUDE_CODE_MAX_RETRIES`를 15로 제한하도록 변경; 무인 세션의 경우 `CLAUDE_CODE_RETRY_WATCHDOG`를 사용
- 배경 subagent를 자동 거부하지 않고 권한 프롬프트를 메인 세션에 표시하도록 변경; 대화에서 어떤 agent가 요청하고 있는지 표시하며, Esc는 해당 도구만 거부함
- `/review <pr>`를 `/code-review medium`과 동일한 review engine을 사용하도록 변경
