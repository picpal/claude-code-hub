## 변경사항

- `/model` picker는 이제 `ANTHROPIC_BASE_URL`이 Anthropic 호환 gateway를 가리킬 때 gateway의 `/v1/models` endpoint에서 모델을 나열합니다
- `claude project purge [path]`를 추가하여 프로젝트의 모든 Claude Code 상태(transcripts, tasks, file history, config entry)를 삭제합니다 — `--dry-run`, `-y/--yes`, `-i/--interactive`, 및 `--all`을 지원합니다
- `--dangerously-skip-permissions`는 이제 `.claude/`, `.git/`, `.vscode/`, shell config 파일 및 기타 이전에 보호되던 경로에 대한 쓰기 프롬프트를 우회합니다(파괴적 제거 명령은 안전장치로 계속 프롬프트를 표시합니다)
- `claude auth login`은 이제 browser callback이 localhost에 도달할 수 없을 때(WSL2, SSH, containers) terminal에 붙여넣은 OAuth code를 허용합니다
- `claude_code.skill_activated` OpenTelemetry event는 이제 사용자가 입력한 slash command에 대해 발생하며 새로운 `invocation_trigger` attribute(`"user-slash"`, `"claude-proactive"`, 또는 `"nested-skill"`)를 전달합니다
- Auto mode: spinner는 이제 permission check가 정체되었을 때 빨간색으로 변하므로 도구가 실행 중인 것처럼 보이지 않습니다
- Host에서 관리하는 deployments(`CLAUDE_CODE_PROVIDER_MANAGED_BY_HOST`)는 더 이상 Bedrock/Vertex/Foundry에서 analytics를 자동으로 비활성화하지 않습니다
- Windows: Microsoft Store를 통해 설치된 PowerShell 7, PATH 없는 MSI, 또는 `.NET global tool`이 이제 감지됩니다
- Windows: PowerShell tool이 활성화되면 Claude는 이제 Bash로 기본 설정하는 대신 PowerShell을 primary shell로 취급합니다
- Read tool: legacy model에서 가양성 거부 및 "this is not malware" commentary를 유발할 수 있는 파일별 malware-assessment reminder를 제거했습니다
- **보안:** `allowManagedDomainsOnly` / `allowManagedReadPathsOnly`가 상위 우선순위의 managed-settings source에 `sandbox` block이 없을 때 무시되는 문제를 수정했습니다
- 2000px보다 큰 이미지 붙여넣기가 session을 깨뜨리는 문제를 수정했습니다 — 이제 이미지가 붙여넣기 시 downscale되며, history의 oversized image는 자동으로 제거되고 request는 다시 시도됩니다
- "OAuth not allowed for organization" error에 대해 login screen을 표시하는 문제를 수정했습니다 — 이제 admin에게 문의하도록 하는 guidance를 표시합니다
- OAuth login이 느리거나 proxied된 연결에서 timeout되는 문제, IPv6-only devcontainers에서, 그리고 browser callback이 localhost에 도달할 수 없을 때 failing하는 문제를 수정했습니다
- concurrent credential write가 유효한 OAuth refresh token을 지울 수 있는 드문 race 조건을 수정했습니다
- API retry countdown이 시도 간 countdown하는 대신 "0s"에 고착되는 문제를 수정했습니다
- Mac이 request 중간에 sleep에서 깨어난 후 "Stream idle timeout" error가 발생하는 문제를 수정했습니다
- Background 및 remote session이 long model thinking pause 중 "Stream idle timeout"으로 거짓으로 abort되는 문제를 수정했습니다
- assistant가 thinking을 완료할 수 있지만 empty turn 실행 후 output을 표시하지 않는 hang을 수정했습니다
- Cursor 및 VS Code 1.92–1.104 integrated terminal에서 과도하게 빠른 trackpad scrolling을 수정했습니다
- claude.ai MCP connector가 needs-auth state에 stuck된 manual server에 의해 suppressed되는 문제를 수정했습니다
- Windows의 no-flicker mode에서 Japanese/Korean/Chinese 텍스트가 garbled character로 rendering되는 문제를 수정했습니다
- `Ctrl+L`이 prompt input을 clearing하는 문제를 수정했습니다 — 이제 readline 동작과 일치하도록 screen redraw만 강제합니다
- Deferred tool(WebSearch, WebFetch 등)이 `context: fork` 및 첫 번째 turn의 다른 subagent를 포함하는 skill에 대해 사용 불가능하던 문제를 수정했습니다
- Plan-mode tool이 `--channels`로 launched된 interactive session에서 사용 불가능한 문제를 수정했습니다
- `/plugin` Uninstall이 "Enabled" 대신 "Uninstalled"를 reporting하던 문제를 수정했습니다
- Linter가 한 번에 많은 파일을 touch할 때 file-modified reminder의 total size를 bounded했습니다
- `/remote-control` retry가 "connecting…"에 stuck되어 있는 것처럼 나타나는 문제를 수정했습니다 — 이제 각 retry는 result를 표시합니다
- Remote Control failure notification이 initial connection failure에 대해 error reason을 표시하지 않는 문제를 수정했습니다
- Windows: clipboard write는 더 이상 EDR/SIEM telemetry에 표시되는 process command-line argument에서 copied content를 노출하지 않습니다. 또한 >22KB selection이 clipboard에 도달하지 못하던 문제도 수정합니다
- PowerShell tool: bare `--`(예: `git diff -- file`)는 더 이상 `--%` stop-parsing token으로 mis-flagged되지 않습니다
- Model이 parallel tool call batch에서 malformed tool name을 emit할 때 Agent SDK hang이 발생하는 문제를 수정했습니다
