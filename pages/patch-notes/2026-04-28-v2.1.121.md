> 원문: [Claude Code v2.1.121 Release Notes](https://github.com/anthropics/claude-code/releases/tag/v2.1.121)

## 변경사항

- MCP server config에 `alwaysLoad` 옵션을 추가했습니다 — `true`로 설정하면 해당 서버의 모든 tool이 tool-search deferral을 건너뛰고 항상 사용 가능합니다
- `claude plugin prune`을 추가하여 orphaned auto-installed plugin dependencies를 제거합니다. `plugin uninstall --prune`도 연쇄적으로 작동합니다
- `/skills`에 type-to-filter search box를 추가하여 긴 목록에서 스크롤 없이 skill을 찾을 수 있습니다
- PostToolUse hooks가 이제 `hookSpecificOutput.updatedToolOutput`을 통해 모든 tool에 대해 tool output을 교체할 수 있습니다 (이전에는 MCP 전용)
- Fullscreen mode: prompt에 입력할 때 이전 출력을 읽기 위해 스크롤을 올린 후에도 더 이상 스크롤이 맨 아래로 돌아가지 않습니다
- Terminal을 초과하는 dialog는 이제 arrow keys, PgUp/PgDn, home/end, mouse wheel을 사용하여 fullscreen 및 non-fullscreen 모드에서 모두 스크롤할 수 있습니다
- Fullscreen mode에서 여러 행에 걸쳐 감싼 긴 URL의 모든 행을 클릭하면 전체 URL이 열립니다
- SDK 및 `claude -p`: `CLAUDE_CODE_FORK_SUBAGENT=1`이 이제 non-interactive sessions에서도 작동합니다
- `--dangerously-skip-permissions`이 더 이상 `.claude/skills/`, `.claude/agents/`, `.claude/commands/` 쓰기에 대해 prompt을 표시하지 않습니다
- `/terminal-setup`이 이제 iTerm2의 "Applications in terminal may access clipboard" 설정을 활성화하여 tmux를 포함한 `/copy`가 작동하도록 합니다
- Startup 중에 transient error가 발생한 MCP servers가 이제 disconnect 상태로 유지되지 않고 최대 3회까지 자동으로 재시도합니다
- Terminal tab session title이 이제 configured `language` 설정으로 생성됩니다
- 동일한 upstream URL을 가진 Claude.ai connectors가 이제 중복으로 나타나지 않고 중복 제거됩니다
- Vertex AI: X.509 certificate-based Workload Identity Federation (mTLS ADC)을 지원합니다
- Upgrade 후 빠른 startup: release-notes splash에서 Recent Activity panel을 제거했습니다
- LSP diagnostic summaries가 이제 클릭/ctrl+o에서 확장되고 expand hint를 표시합니다
- SDK: `mcp_authenticate`이 이제 custom scheme completion 및 claude.ai connectors을 위한 `redirectUri`을 지원합니다
- OpenTelemetry: LLM request spans에 `stop_reason`, `gen_ai.response.finish_reasons`, `user_system_prompt` (`OTEL_LOG_USER_PROMPTS` 뒤에 gated)를 추가했습니다
- [VSCode] Voice dictation이 이제 Claude Code language가 configured되지 않았을 때 `accessibility.voice.speechLanguage` 설정을 존중합니다
- [VSCode] `/context`가 이제 native token usage dialog를 엽니다
- Session에서 많은 이미지를 처리할 때 unbounded memory growth (multi-GB RSS)를 수정했습니다
- 큰 transcript histories를 가진 머신에서 `/usage`가 최대 ~2GB의 메모리를 누수하는 것을 수정했습니다
- Long-running tools가 명확한 progress event를 내보내지 못할 때 memory leak을 수정했습니다
- Claude가 시작된 디렉토리가 mid-session에 삭제되거나 이동될 때 Bash tool이 영구적으로 사용 불가능해지는 것을 수정했습니다
- `--resume`이 external builds에서 startup 시 crash하는 것을 수정했습니다
- `--resume`이 unclean shutdown에 의해 transcript line이 손상된 large sessions에서 실패하는 것을 수정했습니다 — corrupt line이 이제 건너뜁니다
- Bedrock application inference profile ARNs을 사용할 때 `thinking.type.enabled is not supported` error를 수정했습니다
- Microsoft 365 MCP OAuth가 duplicate 또는 unsupported `prompt` parameter로 실패하는 것을 수정했습니다
- Tmux, GNOME Terminal, Windows Terminal, Konsole에서 non-fullscreen mode에서 Ctrl+L을 누르거나 redraw를 트리거할 때 scrollback duplication을 수정했습니다
- Connector-list fetch가 startup에서 transient auth error를 hit할 때 claude.ai MCP connectors가 조용히 사라지는 것을 수정했습니다
- Remote sessions에서 built-in tools에 대한 "Always allow" rules이 worker restarts에서 surviving되지 않는 것을 수정했습니다
- Native build에서 `managed-settings.json`을 통해 설정할 때 `NO_PROXY`가 모든 HTTP clients에 대해 respect되지 않는 것을 수정했습니다
- Managed settings approval prompt가 수락되었을 때에도 session을 exit하는 것을 수정했습니다 — 이제 settings를 적용하고 계속합니다
- `/usage`가 stale OAuth token 후 "rate limited"을 반환하는 것을 수정했습니다 — 이제 자동으로 refresh합니다
- `settings.json`의 invalid legacy enum values가 전체 settings file을 invalidate하는 것을 수정했습니다
- No-flicker mode가 off일 때 `/usage` dialog content가 clipped되는 것을 수정했습니다
- Fullscreen renderer가 off일 때 `/focus`가 "Unknown command"를 표시하는 것을 수정했습니다 — 이제 enable하는 방법을 설명합니다
- Running binary가 mid-session에 삭제될 때 embedded grep/find/rg shell wrappers가 실패하는 것을 수정했습니다 — 이제 installed tools로 fallback합니다
- Large directory trees에서 Bash tool의 `find` 중에 peak file descriptor usage를 감소시켰습니다
