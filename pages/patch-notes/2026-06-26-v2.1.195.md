## 변경사항

- Mouse click/drag/hover를 비활성화하면서 wheel scroll은 유지하는 `CLAUDE_CODE_DISABLE_MOUSE_CLICKS`를 추가했습니다 (fullscreen 모드).
- Hyphenated identifier를 포함한 hook matcher를 수정했습니다 (예: `code-reviewer`, `mcp__brave-search`). 이제 부분 문자열 일치 대신 정확히 일치합니다. Hyphenated MCP server의 모든 tool을 일치시키려면 `mcp__brave-search__.*`를 사용하세요.
- macOS에서 voice dictation이 기본 입력 장치가 변경된 후 장시간 실행되는 세션에서 침묵을 캡처하는 문제를 수정했습니다.
- 공백이 없는 언어(일본어, 중국어, 태국어)의 voice dictation auto-submit이 작동하지 않는 문제를 수정했습니다.
- Project `.claude/settings.json`에 의해서만 활성화된 external plugin이 모든 loader 경로에서 명시적 설치 동의를 요구하지 않는 문제를 수정했습니다.
- Plugin의 `plugin.json` `name`이 마켓플레이스 항목 이름과 다를 때 `/plugin` Enable/Disable이 작동하지 않는 문제를 수정했습니다.
- 더 최신의 Claude Code 버전에서 작성한 경우 `claude agents`에서 background job이 사라지거나 데이터를 잃는 문제를 수정했습니다.
- 충돌한 background task를 다시 열 때 5초 동안 빈 화면 대신 재시작을 표시하는 문제를 수정했습니다.
- Control socket이 시작되지 못하여 재시작을 차단하는 경우 background agent daemon이 도달 불가능한 상태로 실행되는 문제를 수정했습니다.
- Linux에서 voice mode를 개선했습니다. 이제 SoX가 있지만 오디오 캡처 장치가 없는 경우 "no microphone"과 "SoX not installed"를 구분합니다.
- Container가 시작되는 동안 프로비저닝 체크리스트를 통해 Remote session 시작을 개선했습니다.
- `claude agents` 완료된 목록을 개선하여 사용 가능한 수직 공간을 채우도록 했습니다. 짧은 터미널에서는 헤더가 축소되어 라이브 세션이 표시됩니다.
