## 변경된 사항
- API 오류(요청 제한, 인증 실패 등)로 인해 턴이 종료될 때 발생되는 `StopFailure` 훅 이벤트를 추가했습니다.
- 플러그인 업데이트 후에도 유지되는 플러그인 영구 상태를 위한 `${CLAUDE_PLUGIN_DATA}` 변수를 추가했습니다. `/plugin uninstall` 명령을 실행하면 삭제 전에 확인 메시지가 표시됩니다.
- 플러그인과 함께 제공되는 에이전트에 대해 `effort`, `maxTurns`, `disallowedTools` 프론트매터 지원을 추가했습니다
- `set -g allow-passthrough on` 옵션을 사용하여 tmux 내에서 실행할 때 터미널 알림(iTerm2/Kitty/Ghostty 팝업, 진행률 표시줄)이 외부 터미널로 전송됩니다.
- 응답 텍스트가 생성되는 대로 한 줄씩 표시됩니다
- 리눅스의 샌드박스화된 Bash 환경에서 `git log HEAD` 명령어가 "ambiguous argument" 오류로 실패하는 문제를 수정했으며, 작업 디렉터리 내의 스텁 파일이 `git status` 결과를 오염시키는 문제도 해결했습니다.
- 서브에이전트를 사용하는 대용량 세션(5MB 초과)에서 `cc log` 및 `--resume` 명령어가 대화 내역을 아무런 경고 없이 잘라내던 문제를 수정했습니다.
- API 오류로 인해 스톱 훅이 트리거되고, 이로 인해 차단 오류가 모델로 다시 전달되어 무한 루프가 발생하는 문제를 수정했습니다.
- `deny: ["mcp__servername"]` 권한 규칙이 모델로 전송되기 전에 MCP 서버 도구를 제거하지 않아, 모델이 차단된 도구를 확인하고 사용하려고 시도할 수 있었던 문제를 수정했습니다.
- 절대 경로에서 `sandbox.filesystem.allowWrite`가 작동하지 않던 문제를 수정했습니다(이전에는 `//` 접두사가 필요했습니다).
- macOS에서 `/sandbox`의 ‘종속성’ 탭에 macOS 전용 정보 대신 Linux 필수 구성 요소가 표시되던 문제를 수정했습니다.
- **보안:** `sandbox.enabled: true`가 설정되어 있으나 종속성이 누락된 경우 샌드박스가 아무런 경고 없이 비활성화되던 문제를 수정했습니다. 이제 시작 시 눈에 띄는 경고 메시지가 표시됩니다.
- `bypassPermissions` 모드에서 `.git`, `.claude` 및 기타 보호된 디렉터리가 확인 메시지 없이 쓰기 권한이 부여되던 문제를 수정했습니다.
- 일반 모드에서 Ctrl+U를 누르면 readline의 줄 삭제 기능이 아닌 스크롤이 작동하도록 수정함 (Ctrl+U/Ctrl+D로 반 페이지 스크롤하는 기능은 트랜스크립트 모드에서만 사용 가능하도록 변경됨)
- 음성 모드 수정자-조합 푸시-투-톡(PTT) 키 바인딩(예: Ctrl+K)이 즉시 활성화되지 않고 키를 길게 눌러야 하는 문제를 수정했습니다.
- WSLg(Windows 11) 환경에서 WSL2의 음성 모드가 작동하지 않던 문제를 수정했습니다. 이제 WSL1/Win10 사용자에게는 명확한 오류 메시지가 표시됩니다.
- `--worktree` 플래그를 사용할 때 워크트리 디렉터리에서 스킬과 훅이 로드되지 않던 문제를 수정했습니다.
- `CLAUDE_CODE_DISABLE_GIT_INSTRUCTIONS` 및 `includeGitInstructions` 설정이 시스템 프롬프트의 git status 섹션을 숨기지 않던 문제를 수정했습니다.
- Dock 또는 Spotlight에서 VS Code를 실행할 때 Bash 도구가 Homebrew 및 기타 PATH에 의존하는 바이너리를 찾지 못하는 문제를 수정했습니다.
- 트루컬러(truecolor) 지원을 명시하지 않는 VS Code/Cursor/code-server 터미널에서 클로드 오렌지 색상이 흐릿하게 표시되던 문제를 수정했습니다.
- `/model` 선택기에 사용자 정의 항목을 추가하기 위해 `ANTHROPIC_CUSTOM_MODEL_OPTION` 환경 변수를 추가했으며, 표시를 위해 선택적으로 `_NAME` 및 `_DESCRIPTION` 접미사가 붙은 변수를 사용할 수 있습니다.
- Haiku 모델을 사용할 때 `ANTHROPIC_BETAS` 환경 변수가 아무런 오류 메시지 없이 무시되던 문제를 수정했습니다
- 대기 중인 프롬프트가 줄바꿈 구분자 없이 연결되는 문제를 수정했습니다
- 대용량 세션을 재개할 때 메모리 사용량과 시작 시간이 개선되었습니다
- [VSCode] 이미 인증된 상태에서 사이드바를 열 때 로그인 화면이 잠깐 깜빡이는 현상을 수정했습니다
- [VSCode] Opus 선택 시 발생하던 "API 오류: 요청 제한 초과" 문제를 수정했습니다. 이제 요금제 등급이 확인되지 않은 구독자에게는 모델 드롭다운 메뉴에 1M 컨텍스트 변형 옵션이 더 이상 표시되지 않습니다.
