## 변경사항

- Claude in Chrome이 이제 일반 공급됨
- `claude agents`에 background agent notifications 추가 — 입력이 필요하거나 완료된 세션이 이제 `Notification` hook(`agent_needs_input` / `agent_completed`)을 발생시킴
- chart 및 dashboard 디자인 가이던스를 위한 `/dataviz` skill 추가, 실행 가능한 color-palette validator 포함
- Gateway: Claude Platform on AWS(anthropicAws)를 upstream provider로 추가; model-not-found 응답이 이제 failover chain을 진행함
- `claude agents`에서 시작된 background agents는 이제 worktree에서 코드 작업을 마칠 때 입력을 요청하는 대신 commit, push, draft PR을 열 수 있음
- 기본 제공되는 Explore agent는 이제 haiku에서 실행되는 대신 main session의 model을 상속받음(opus로 제한)
- Subagents 및 context compaction은 이제 session의 extended thinking configuration을 상속받으므로 위임된 작업의 출력 품질이 향상됨
- mid-response 중 간헐적인 네트워크 끊김으로 인해 turn이 중단되는 문제 수정 — ECONNRESET 같은 일시적 오류가 이제 실패 대신 backoff를 사용하여 재시도됨
- sandboxed processes가 반복적으로 동일한 network host에 접근할 때 excessive background classifier requests 문제 수정
- web, desktop, 및 VS Code task panels의 background tasks가 완료된 후 또는 session을 재개한 후 "Running"에서 stuck되는 문제 수정
- Agent teams 수정: API 오류로 인해 die된 teammate는 이제 lead에 "failed"를 보고하고, stuck teammate에 메시지를 보내면 즉시 깨어나 재시도됨
- STS token이 만료될 때 Claude Platform on AWS 및 Mantle sessions이 "Please run /login"으로 dead-ending되는 문제 수정 — `awsAuthRefresh`는 이제 자동으로 실행됨
- macOS background agent sessions에서 local-network hosts에 대한 "no route to host" 문제 수정, Local Network entitlements 선언
- worktree를 입력하고 나간 후 `/desktop`이 "Cannot determine working directory"로 실패하는 문제 수정
- agents view가 열려 있을 때 macOS에서 background agents가 ~52초마다 반복적으로 "Reconnecting…"을 표시하는 문제 수정
- `claude attach <id>` 내에서 `←` 키를 누를 때 agent view를 열지 않고 shell로 나가는 문제 수정
- `claude --bg`가 `--print`/`-p`와 결합될 때 attach 불가능한 session을 자동으로 생성하는 문제 수정; 충돌하는 flag은 이제 미리 거부됨
- workflow progress view가 SDK 및 desktop-app sessions에서 phase counter가 정확하게 유지되는 동안 목록에서 가장 오래된 agents를 드롭하는 문제 수정
- symlinked path를 통해 도달한 파일일 때 `.claude/rules/` conditional rules이 로드되지 않는 문제 수정
- macOS의 Warp에서 fullscreen mode에서 Cmd+click이 URLs를 열지 않는 문제 수정
- fullscreen mode에서 double-click word selection이 scheme을 포함한 전체 URL을 선택하도록 수정
- plan mode가 session이 plan mode로 시작될 때 read-only tool calls을 자동으로 허용하지 않는 문제 수정
- `/branch`가 첫 번째 real prompt 대신 compaction summary에서 기본 fork name을 파생시키는 문제 수정
- Focus mode 개선: turn에서 시작된 subagents가 이제 activity summary에 나타나고, completed background notifications이 단일 count로 fold됨
- highlight.js 11로 업그레이드하여 code blocks, diffs, 및 file previews에서 syntax highlighting accuracy 개선
- Keyboard shortcut hints는 이제 SSH를 통해 Mac에서 연결할 때 alt/super 대신 opt/cmd를 표시함
- API retry UX 개선: 두 번째 시도 후 오류 이유가 표시되고, API가 overloaded될 때 spinner tip을 status page link로 대체함
- `/login`이 이제 "사용 불가능하다"고 말하는 대신 `claude agents` view에서 sign-in dialog를 열 수 있음
- Subagents는 이제 이들을 시작한 agent의 메시지를 일반적인 task direction으로 취급함; agent의 메시지는 여전히 사용자의 승인으로 취급되지 않음
- `/agents` wizard 제거; Claude에 subagents를 만들거나 관리하도록 요청하거나 `.claude/agents/`를 직접 편집함
