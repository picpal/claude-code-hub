## 변경사항

- `/config key=value` 문법을 추가하여 프롬프트에서 모든 설정을 지정할 수 있도록 함 (예: `/config thinking=false`) — interactive, `-p`, Remote Control에서 작동
- `sandbox.allowAppleEvents` opt-in 설정을 추가하여 macOS에서 sandboxed 명령이 Apple Events를 전송할 수 있도록 함
- `CLAUDE_CLIENT_PRESENCE_FILE` environment variable을 추가함: 마커 파일을 지정하여 컴퓨터에 있을 때 모바일 푸시 알림을 억제
- 번들된 Bun runtime을 1.4로 업그레이드
- 긴 문단의 스트리밍 개선: 첫 번째 줄 바꿈을 기다리지 않고 한 줄씩 텍스트가 표시됨
- 자동 재시도 개선: thinking 중 API 연결이 끊어지면 "Connection closed while thinking" 표시 대신 자동으로 재시도
- subagent 패널 개선: idle subagent는 30초 후 자동 숨김, 목록은 5줄로 제한되며 스크롤 힌트 표시, 키보드 힌트는 이제 footer에 표시
- MCP OAuth 브라우저 페이지를 Claude Code의 비주얼 스타일과 일치하도록 개선하고 성공 시 자동 종료
- fullscreen mode URL 열기를 Cmd+click (macOS) / Ctrl+click으로 변경하여 기본 terminal 동작과 일치
- `Improved N memories` 줄이 verbose mode 외부에서 개별 파일을 더 이상 나열하지 않도록 변경
- custom `ANTHROPIC_BASE_URL` 및 Foundry에서 prompt caching을 읽지 못하는 문제 수정 (매 차례마다 변경되는 per-request attestation token 때문)
- 네트워크 드라이브 및 클라우드 동기화 폴더에서 Write/Edit가 0-byte 또는 truncated 파일을 생성하는 문제 수정
- macOS에서 Apple Events entitlement을 추가하여 `open`, `osascript`, 브라우저 기반 auth flow가 error -600으로 실패하는 문제 수정
- 시작 regression 수정 (fresh 환경에서 시작당 약 120ms, 2.1.169에서 도입): MCP server가 구성되지 않았을 때 첫 프롬프트가 더 이상 managed-settings fetch를 기다리지 않음
- degraded network에서 account settings fetch가 느릴 때 시작이 최대 15초간 blank terminal로 차단되는 문제 수정
- `.claude.json`에 corrupted null project 항목이 포함되어 있을 때 시작 crash (`TypeError: Cannot read properties of null`) 수정
- Spotlight이 reindexing 중일 때 macOS TUI가 세션 시작에서 freezing 되는 문제 수정 (Ctrl+C 응답 없음)
- 다른 Claude Code 프로세스가 30일 transcript cleanup을 실행했을 때 오래 실행되는 idle 세션이 history를 잃는 문제 수정
- foreground subagent가 unbounded nested chain을 생성하는 문제 수정; 이제 background subagent와 동일한 5-level depth limit을 준수
- `/recap` 및 conversation fork가 model switch 직후 이전 model을 사용하는 문제 수정
- subagent "Thinking" 지속시간이 subagent 자신의 경과 시간이 아닌 parent agent의 경과 시간을 표시하는 문제 수정
- nested agent에서 차단된 subagent가 agent panel에서 "waiting" 대신 ticking elapsed time을 표시하는 문제 수정
- API retry indicator ("Retrying in 0s · attempt N/10")가 retry 성공 후 화면에 남아있는 문제 수정
- 짧은 remaining lifetime을 가진 AWS `awsCredentialExport` credentials이 매분 credential refresh를 유발하고, 이제 `aws configure export-credentials`의 JSON shape을 허용하는 문제 수정
- `claude mcp get`/`list`가 tools/list 실패 시 `✓ Connected`를 표시하는 문제 수정; 이제 `! Connected · tools fetch failed`를 error detail과 함께 표시
- `/remote-control`이 stale "connecting…" 줄을 남기는 문제 수정; 이제 연결되면 transcript에서 확인
- ExitWorktree가 bare `git`을 Windows에서 해석할 수 없을 때 clean worktree를 "Could not verify worktree state" 메시지로 제거하기를 거부하는 문제 수정
- `~/.claude/settings.json`이 symlinked `~/.claude` 아래의 relative symlink일 때 settings 변경 (예: `/effort` 또는 `/model`)이 ENOENT로 실패하는 문제 수정
- IDE selection line number가 context reminder에서 off by one인 문제 수정 (IntelliJ 및 VS Code)
- native terminal selection (modifier+drag) 후 fullscreen에서 Ctrl+C가 앱의 이전 selection으로 clipboard를 덮어쓰는 문제 수정
- clipboard에 text가 포함되어 있을 때 Ctrl+V가 붙여넣기 대신 "No image found in clipboard"를 표시하는 문제 수정
- agents directory가 이미 존재할 때 agent 생성이 "EEXIST: file already exists"로 실패하는 문제 수정 (Windows/OneDrive)
- AskUserQuestion preview content가 dialog edge에서 잘리고 word-wrap하지 않는 문제 수정
- AskUserQuestion multi-select question이 제출 시 typed "Other" free-text 답변을 자동으로 삭제하는 문제 수정
- `/stats` "Most active day" 및 daily token chart 날짜가 UTC-negative timezone에서 하루 일찍 표시되는 문제 수정
- Linux에서 `/copy` 및 copy-on-select가 Claude Code 시작 후 설치된 clipboard utility를 감지하지 못하는 문제 수정
- tab-indented code가 Write (create-file) preview에서 잘못된 indentation으로 렌더링되는 문제 수정
- user prompt가 mid-turn에 queued 될 때 transcript에서 full-width background highlight를 표시하지 않는 문제 수정
- Ghostty에서 activity spinner의 pulse가 잘못된 glyph size에서 dwell하는 문제 수정
