## 변경사항

- 전체 화면 모드에서 마우스 휠 스크롤 가속을 비활성화하는 `wheelScrollAccelerationEnabled` 설정이 추가되었습니다
- `/model` 선택 도구가 Default로 확인되는 모델 제품군을 숨기는 문제가 수정되었습니다 — Max/Team Premium/Enterprise 플랜에서는 Opus가 자체 행으로 나타나고, Pro/Team 플랜에서는 Sonnet이, pay-as-you-go API 계정에서는 Opus가 표시됩니다
- `ANTHROPIC_DEFAULT_SONNET_MODEL`이 다른 Sonnet을 고정할 때 `/model` 선택 도구가 하드코딩된 Sonnet 버전 레이블을 표시하는 문제가 수정되었습니다
- "Fable 5는 이제 사용 크레딧을 소비하고 있습니다" 배너가 사용량 기반 청구를 사용하는 엔터프라이즈 계정에 대해 잘못 표시되는 문제가 수정되었습니다
- Bedrock GovCloud 리전(`us-gov-*`)이 잘못된 inference profile 접두사(`global` 대신 `us-gov`)를 생성하여 파생된 모델 ID에서 400 오류가 발생하는 문제가 수정되었습니다
- 백그라운드 세션이 백그라운드 daemon을 시작한 셸에서 다른 세션의 `ANTHROPIC_*` provider 환경 변수(gateway URL, 사용자 정의 헤더, `/model` 별칭)를 상속하는 문제가 수정되었습니다
- macOS 및 Linux에서 셸 명령이 중단되거나 종료된 직후 Claude Code를 종료할 때 1-2초 지연이 발생하는 문제가 수정되었습니다
- git commit co-author 속성이 일부 모델에 대해 잘못된 모델 이름을 표시하는 문제가 수정되었습니다
- `/advisor` 대화 상자가 `availableModels` 허용 목록으로 차단된 저장된 advisor 모델을 사전 선택하는 문제가 수정되었습니다
- Skill hot-reload가 단일 skill이 변경될 때 전체 skill 목록을 다시 전송하는 문제가 수정되었습니다. 이제 변경된 skill만 다시 알려집니다
- Workflow tool `agent()` subagents에서 per-agent attribution 헤더가 누락되는 문제가 수정되었습니다
- [VSCode] 계정 및 사용 대화 상자(`/usage`)에 지난 24시간 또는 7일 동안의 cache misses, long context, subagents 및 per-skill/agent/plugin/MCP 분석을 보여주는 사용 attribution이 추가되었습니다
- 유휴 상태로 대기한 후 회수될 때 pre-warmed 백그라운드 workers가 "Could not resolve authentication method" 오류로 실패하는 문제가 수정되었습니다
