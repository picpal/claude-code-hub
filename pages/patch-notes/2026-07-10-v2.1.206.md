## 변경사항

- `/cd`에 디렉토리 경로 제안 추가됨, `/add-dir` 동작과 일치
- checked-in `CLAUDE.md` 파일을 코드베이스에서 파생될 수 있는 내용을 제거하여 트리밍하도록 제안하는 `/doctor` 체크 추가됨
- `/commit-push-pr`이 이제 `origin`뿐만 아니라 저장소의 설정된 푸시 remote(`remote.pushDefault` 또는 단일 remote만 구성되어 있을 때의 유일한 remote)에 대해 `git push`를 자동 허용
- Gateway: `/login`이 이제 Anthropic 운영 공용 gateway endpoints 지원
- `EnterWorktree`가 이제 프로젝트의 `.claude/worktrees/` 디렉토리 외부의 git worktree에 진입하기 전에 확인 요청
- Background agents가 이제 attach할 때 느린 stale-session 업그레이드를 기다리는 대신 Claude Code 업데이트 직후 백그라운드에서 새 버전으로 업그레이드
- 만료된 login이 `/login` 실행을 prompt하는 대신 모든 model에서 오해의 여지가 있는 "There's an issue with the selected model" 오류를 일으키는 문제 수정됨
- `claude --resume` 및 `--continue`가 시작 시 키보드 입력에 응답하지 않는 문제 수정됨
- `--mcp-config` 또는 `.mcp.json`을 통해 설정된 MCP servers가 per-server `request_timeout_ms`를 무시하여 오래 실행되는 MCP tool calls이 fresh sessions에서 60초 기본값으로 시간 초과되는 문제 수정됨
- `claude agents` / `--bg` background workers에서 `CLAUDE_CODE_EXTRA_BODY`가 조용히 무시되는 문제 수정됨; shell로 내보낸 override가 이제 dispatching session을 따름
- OAuth MCP servers가 단일 실패한 token refresh 후 수동 재인증을 요구하는 문제 수정됨
- `--permission-prompt-tool`이 MCP server를 가리키며 server가 연결을 완료하기 전의 cold start에서 "MCP tool not found"으로 크래시되는 문제 수정됨
- `/model` picker rows가 행 이름과 다른 model의 가격을 인쇄하는 문제, 그리고 billing하지 않는 providers의 first-party list prices를 인용하는 문제 수정됨
- entitlement 또는 allowlist 제한이 위치된 행을 제거할 때 server 제공 model rows가 `/model` picker에서 잘못 배치되는 문제 수정됨
- desktop sessions이 slash command가 mid-turn에서 전송된 후 "running"을 표시하는 동안 stuck되는 문제 수정됨
- Windows의 bare `claude --resume` 전에 setup prompt가 나타났을 때 agents view에서 키보드 입력이 무시되는 문제 수정됨
- `claude rm`이 removed job을 daemon roster에 남겨두어 행이 `claude agents`에서 다시 나타나는 문제 수정됨
- `/remote-control`이 logged out일 때 "Unknown command"를 표시하는 문제 수정됨 — 이제 sign in 방법 설명
- left arrow가 workflow detail view의 phase 또는 agent에서 물러나지 않는 문제 수정됨
- `/status`가 동일한 broken-install warning을 두 번 나열하는 문제 수정됨
- LSP plugins에 대한 false "disused plugin" tips 및 skewed disuse telemetry 수정됨
- `/doctor`의 update check이 settings channel 대신 Homebrew installs를 해당 cask의 channel과 비교하도록 수정됨
- fullscreen jump-to-bottom pill이 macOS에서 Ctrl+End를 제안하고, rebound chords를 표시하지 않으며, transcript를 래핑하는 문제 수정됨
- Bedrock: restricted egress가 있는 networks에서 `awsCredentialExport` helper를 사용할 때 다중 분 시작 hang 수정됨
- `/code-review` findings quality가 claude-opus-4-8에서 모든 effort levels에 걸쳐 개선됨
- agents view 개선됨: status column이 이제 64자로 truncate하는 대신 전체 terminal width 사용
- agents view 변경됨: Ctrl+X이 이제 completed session을 영구적으로 제거하며, sessions이 더 이상 두 번 렌더링되지 않음; deleted background jobs이 deleted 상태 유지
