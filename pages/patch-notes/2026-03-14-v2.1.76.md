## 변경된 사항
- MCP 유도 기능 추가 — 이제 MCP 서버는 대화형 대화상자(양식 필드 또는 브라우저 URL)를 통해 작업 도중 구조화된 입력을 요청할 수 있습니다.
- 응답이 반환되기 전에 이를 가로채거나 재정의할 수 있도록 새로운 `Elicitation` 및 `ElicitationResult` 훅을 추가했습니다
- 시작 시 세션의 표시 이름을 설정할 수 있는 `-n` / `--name <name>` CLI 플래그를 추가했습니다
- 대규모 모노리포지토리에서 `claude --worktree` 명령을 사용할 때, git sparse-checkout을 통해 필요한 디렉터리만 체크아웃할 수 있도록 `worktree.sparsePaths` 설정을 추가했습니다.
- 압축이 완료된 후 실행되는 `PostCompact` 훅을 추가했습니다
- 모델의 노력 수준을 설정하는 `/effort` 슬래시 명령어를 추가했습니다
- 세션 품질 설문조사 기능 추가 — 기업 관리자는 `feedbackSurveyRate` 설정을 통해 표본 조사 빈도를 구성할 수 있습니다
- 대화 압축 후 `ToolSearch`를 통해 로드된 지연 도구(deferred tools)의 입력 스키마가 손실되어, 배열 및 숫자 매개변수가 유형 오류로 인해 거부되던 문제를 수정했습니다.
- 슬래시 명령어 실행 시 "알 수 없는 기술"이 표시되던 문제를 수정했습니다.
- 계획이 이미 승인된 후에도 재승인을 요청하던 문제를 수정했습니다
- 권한 대화 상자나 계획 편집기가 열려 있을 때 음성 모드가 키 입력을 인식하지 못하던 문제를 수정했습니다
- npm을 통해 설치했을 때 Windows에서 `/voice` 명령어가 작동하지 않던 문제를 수정했습니다.
- 1MB 컨텍스트 세션에서 `model:` 프론트매터가 포함된 스킬을 호출할 때 발생하는 잘못된 “컨텍스트 제한 초과” 오류를 수정했습니다.
- 비표준 모델 문자열을 사용할 때 발생하는 “이 모델에서는 적응형 사고가 지원되지 않습니다”라는 오류를 수정했습니다
- 따옴표로 묶인 인자에 `#`이 포함된 경우 `Bash(cmd:*)` 권한 규칙이 일치하지 않는 문제를 수정했습니다.
- 파이프 및 복합 명령어의 경우, Bash 권한 대화 상자에서 “다시 묻지 않음”을 선택했을 때 전체 원시 명령어가 표시되던 문제를 수정했습니다.
- 연속적인 실패 후 자동 압축이 무한정 재시도되던 문제를 수정했습니다. 이제 회로 차단기가 3회 시도 후 중단됩니다.
- 재연결에 성공한 후에도 MCP 재연결 로딩 아이콘이 계속 표시되던 문제를 수정했습니다.
- 마켓플레이스 데이터가 동기화되기 전에 LSP 관리자가 초기화될 경우 LSP 플러그인이 서버를 등록하지 못하던 문제를 수정했습니다.
- SSH를 통한 tmux에서 클립보드 복사 기능 수정 — 이제 터미널에 직접 쓰기와 tmux 클립보드 통합 방식을 모두 시도합니다
- 성공 메시지에서 `/export` 명령어가 전체 파일 경로 대신 파일 이름만 표시되던 문제를 수정했습니다.
- 텍스트를 선택한 후 새 메시지로 대본이 자동으로 스크롤되지 않던 문제를 수정했습니다
- 로그인 방식 선택 화면에서 Esc 키를 눌러 나갈 수 없던 문제를 수정했습니다.
- 여러 원격 제어 관련 문제를 수정했습니다: 서버가 유휴 환경을 정리할 때 세션이 아무런 경고 없이 종료되는 문제, 빠른 메시지가 일괄 처리되지 않고 하나씩 대기열에 쌓이는 문제, 그리고 JWT 갱신 후 오래된 작업 항목으로 인해 메시지가 재전송되는 문제를 해결했습니다.
- 장시간 웹소켓 연결이 끊어진 후 브리지 세션이 복구되지 않던 문제를 수정했습니다
- 소프트 숨김 명령어의 정확한 이름을 입력했을 때 슬래시 명령어가 발견되지 않던 문제를 수정했습니다.
- 원격 브랜치가 이미 로컬에 존재하는 경우, git 참조를 직접 읽어들이고 불필요한 `git fetch` 단계를 생략함으로써 `--worktree`의 시작 성능을 개선했습니다.
- 백그라운드 에이전트 동작 개선 — 이제 백그라운드 에이전트를 종료하더라도 대화 컨텍스트에 저장된 부분 결과가 유지됩니다
- 모델 대체 알림 기능 개선 — 이제 상세 모드 뒤에 숨겨지지 않고 항상 표시되며, 이해하기 쉬운 모델 이름이 사용됩니다
- 어두운 터미널 테마에서 인용문의 가독성이 개선되었습니다. 이제 텍스트가 흐릿하게 표시되는 대신 이탤릭체로 표시되며 왼쪽에 구분선이 추가됩니다.
- 오래된 작업 트리 정리 기능 개선 — 병렬 실행이 중단된 후 남아 있던 작업 트리가 이제 자동으로 정리됩니다
- 원격 제어 세션 제목 개선 — 이제 “대화형 세션”이 표시되던 것에서 벗어나, 사용자의 첫 번째 입력 내용을 기반으로 제목이 생성됩니다.
- `/voice` 명령어를 개선하여, 활성화 시 음성 입력 언어를 표시하고, `language` 설정이 음성 입력에서 지원되지 않을 경우 경고 메시지를 표시하도록 했습니다.
- 하위 명령어를 지원하기 위해 `--plugin-dir` 옵션을 하나의 경로만 허용하도록 업데이트했습니다. 여러 디렉터리를 지정하려면 `--plugin-dir` 옵션을 반복해서 사용하십시오.
- [VSCode] 쉼표가 포함된 gitignore 패턴으로 인해 @멘션 파일 선택기에서 특정 파일 유형 전체가 자동으로 제외되는 문제를 수정했습니다.
