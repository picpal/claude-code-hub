## 변경사항

- Agent tool `subagent_type` 매칭을 개선하여 대소문자 및 구분 기호를 구분하지 않는 값을 허용하도록 함 (예: `"Code Reviewer"`가 `code-reviewer`로 해석됨)
- Agent color palette를 업데이트함
- `disableAllHooks` 또는 `allowManagedHooksOnly`가 설정된 경우 `/goal`이 조용히 멈추던 문제를 수정 — 이제 해석되지 않는 indicator 대신 명확한 메시지를 표시함
- 심볼릭 링크된 settings 파일로 인해 변경 이벤트가 잘못 귀속되고 오탈한 `ConfigChange` hook이 발생하던 settings hot-reload 회귀 문제를 수정함
- background service가 idle-exit를 앞두고 있을 때 `claude --bg`가 "connection dropped mid-request"로 실패하던 문제를 수정함
- enterprise endpoint security가 있는 머신에서 background service 시작이 실패하던 문제를 수정 — 더 많은 시간을 허용함
- remote managed settings이 401에서 재시도하지 않던 문제를 수정 — 이제 force-refreshed token으로 한 번 재시도함
- managed `extraKnownMarketplaces` auto-update policy이 `known_marketplaces.json`에 유지되지 않던 문제를 수정함
- `/loop`이 이미 완료 시 알림을 받는 background task를 폴링하기 위해 중복 wakeup을 스케줄링하던 문제를 수정함
- Windows에서 누락된 executable (예: `gh`)이 모든 check 시마다 동기식 `where.exe` re-spawn을 트리거하던 recurring event-loop stall 문제를 수정함
- `offset`이 공백으로 채워진 문자열 또는 `+` 접두사가 붙은 문자열로 전달될 때 `Read` tool call이 validation에 실패하던 문제를 수정함
- `plugin.json`이 일치하는 키를 설정하기 때문에 기본 component folder (예: `commands/`)가 조용히 무시되던 문제를 수정 — 이제 native terminal cursor이 terminal이 포커스를 잃을 때 input caret에 머물러 있음
- Plugin은 이제 `plugin.json`이 일치하는 키를 설정하기 때문에 기본 component folder (예: `commands/`)이 조용히 무시될 때 경고함. `/doctor`, `claude plugin list`, 및 `/plugin`에서 표시됨.
