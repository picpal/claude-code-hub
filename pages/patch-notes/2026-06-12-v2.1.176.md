> 원문: [Claude Code v2.1.176 Release Notes](https://github.com/anthropics/claude-code/releases/tag/v2.1.176)

## 변경사항

- Session 제목이 이제 대화의 언어로 생성됩니다(`language` 설정을 지정하여 특정 언어를 고정할 수 있습니다)
- footer 행의 regex 매칭 링크 배지를 위한 `footerLinksRegexes` 설정이 추가되었으며, 사용자 또는 관리형 설정을 통해 구성 가능합니다
- Bedrock 자격증명 캐싱 개선: `awsCredentialExport`의 자격증명이 고정된 1시간 대신 `Expiration`까지 캐시됩니다
- `availableModels` 적용 수정: alias 모델 선택이 더 이상 `ANTHROPIC_DEFAULT_*_MODEL` environment 변수를 통해 차단된 모델로 리디렉션될 수 없으며, `/fast`는 이제 허용 목록 외의 모델로 전환하려고 할 때 전환을 거부합니다
- Opus 4.8이 활성화되지 않은 조직의 Fable 5에서 auto mode가 실패하는 문제 수정 — classifier가 이제 사용 가능한 최고의 Opus 모델로 폴백됩니다
- Read/Edit/Write tool 경로에 대한 hook `if` 조건 수정: `Edit(src/**)`, `Read(~/.ssh/**)`, `Read(.env)`과 같은 문서화된 패턴이 이제 올바르게 매칭됩니다
- `.claude/settings.json`이 절대 경로를 가진 symlink일 때 Linux sandbox가 시작되지 않는 문제 수정
- SSH를 통한 tmux 내에서 `/copy` 및 마우스 선택 복사가 시스템 clipboard에 도달하지 못하는 문제와 3.2보다 이전 버전에서 tmux paste buffer가 로드되지 않는 문제 수정
- Remote Control이 web/mobile에서 연결될 때 session의 모델을 무음으로 전환하는 문제 수정
- Remote Control 연결 해제 알림이 숫자 코드 대신 사람이 읽을 수 있는 이유를 표시하지 않는 문제 및 연결 실패가 대화 transcript에 중복 줄을 추가하는 문제 수정
- 다른 계정으로 로그인할 때 Remote Control session이 연결 해제되지 않는 문제 수정
- `/cd` 및 worktree 이동이 session이 이전 디렉토리의 git branch를 보고하도록 남겨두는 문제 수정
- `claude agents`에서 한 창에서 뒤로 가기를 누르면 동일한 session에 연결된 다른 창이 분리되지 않는 문제 수정
- backgrounded session이 `/bg` mid-turn에 계속할 내용이 없을 때 "Working"을 영구적으로 표시하는 문제 수정
- PR URL로 background agent 검색 수정: 예약된 wakeup 중이나 작업이 차단되는 동안 열린 PR이 이제 `claude agents` 검색에 나타납니다
- agents 보기 입력이 Windows에서 텍스트 커서를 표시하지 않는 문제 수정
- `claude --bg -cn <name>`이 session 이름을 seeding하지 않는 문제 수정
- Background session이 respawn 전에 유지된 상태에서 Windows network 경로를 중립화하도록 수정
- Background session respawn이 손상된 state 파일의 malformed resume ID를 거부하는 문제 수정
- `~/.claude/daemon`에 ReadOnly 속성이 있을 때 Windows background service daemon이 시작되지 않는 문제 수정
- 클레임되기 전에 너무 오래 유휴 상태일 때 cloud session이 "Could not resolve authentication method" 오류로 실패하는 문제 수정
- Background session이 이제 auto-update를 통해 창이 열려있을 때 회신을 제출할 수 없을 때 더 명확한 안내를 표시하며, `claude daemon status`가 version-skew 동작을 설명합니다
