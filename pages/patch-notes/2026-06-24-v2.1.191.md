## 변경사항

- `/rewind` 지원이 추가되어 `/clear`가 실행되기 전 대화에서 재개할 수 있게 됨
- 스트리밍 응답 중 이전 출력을 읽는 동안 스크롤 위치가 하단으로 점프하는 문제 수정
- 중지된 후 백그라운드 에이전트가 부활하는 문제 수정 — 작업 패널에서 에이전트를 중지하면 이제 영구적으로 적용됨
- 조직의 정책에 의해 비활성화될 때 `/voice`가 제네릭 "not available" 메시지를 표시하는 문제 수정 — 이제 제한 사항을 설명함
- Windows Terminal에서 줄이 바뀔 때 `/login` URL이 잘려서 열리는 문제 수정
- Ghostty over ssh/tmux에서 전체 화면 모드의 Cmd+click 링크 문제 수정
- `claude agents`가 백그라운드 세션에 `/usage` 같은 builtin slash commands를 힌트 표시 대신 프롬프트 텍스트로 전송하는 문제 수정
- `claude agents` 작업 행이 `[Image #N]` 플레이스홀더 대신 붙여넣은 이미지의 전체 파일 시스템 경로를 표시하는 문제 수정
- 쉼표로 구분된 매처를 사용하는 hooks (예: `"Bash,PowerShell"`)가 자동으로 실행되지 않는 문제 수정
- `/permissions` Recently-denied 탭: 거부를 승인하면 이제 자동으로 삭제되지 않고 닫을 때 유지됨
- 오버플로우 한계를 넘어 명단을 스크롤할 때 에이전트 패널이 한 행씩 점프하는 문제 수정
- 환영 splash art가 기본 80×24 macOS Terminal 창을 초과하는 문제 수정
- 관리 설정 수정: `forceRemoteSettingsRefresh`가 이제 MDM 또는 파일 정책을 통해 설정될 때 적용되며, 가져오기는 프록시가 오래된 응답을 제공하지 않도록 `Cache-Control: no-cache`를 전송함
- Sandbox network 권한 대화 개선: "Yes"로 허용한 호스트가 이제 세션 동안 기억되어 모든 연결마다 다시 묻지 않음
- MCP server 안정성 개선: capability discovery (`tools/list`, `prompts/list`, `resources/list`)가 이제 짧은 백오프로 일시적 네트워크 오류를 재시도함
- MCP OAuth 개선: discovery 및 token 요청이 이제 일시적 네트워크 오류 발생 후 한 번 재시도하며, headless 환경은 브라우저 팝업을 건너뛰고 바로 URL 붙여넣기 프롬프트로 이동함
- MCP 오류 메시지 개선: HTTP 404 오류가 이제 URL을 표시하고 MCP config를 가리킴
- Vim mode prompt-history 검색 (NORMAL `/`) 개선으로 slash commands에 도달하는 방법을 힌트로 표시
- 텍스트 업데이트를 100ms로 통합하여 스트리밍 응답 중 CPU 사용량 ~37% 감소
- 터미널 출력 캐시의 장시간 세션 메모리 증가 감소
