## 변경사항

- CLI를 번들로 제공되는 JavaScript 대신 네이티브 Claude Code binary를 생성하도록 변경했습니다(플랫폼별 선택적 종속성을 통해).
- `sandbox.network.deniedDomains` 설정을 추가하여 더 광범위한 `allowedDomains` wildcard가 허용하더라도 특정 도메인을 차단할 수 있도록 했습니다.
- Fullscreen 모드: Shift+↑/↓를 누르면 이제 선택을 표시 영역을 넘어 확장할 때 viewport를 스크롤합니다.
- `Ctrl+A` 및 `Ctrl+E`는 이제 multiline input에서 현재 logical line의 시작/끝으로 이동하며, readline 동작과 일치합니다.
- Windows: `Ctrl+Backspace`는 이제 이전 단어를 삭제합니다.
- 응답 및 bash output의 긴 URL은 줄을 넘을 때 클릭 가능한 상태를 유지합니다(OSC 8 hyperlinks를 지원하는 터미널에서).
- `/loop` 개선됨: Esc를 누르면 이제 pending wakeup을 취소하고, wakeup은 명확성을 위해 "Claude resuming /loop wakeup"으로 표시됩니다.
- `/extra-usage`는 이제 Remote Control (모바일/웹) 클라이언트에서 작동합니다.
- Remote Control 클라이언트는 이제 `@`-file autocomplete 제안을 조회할 수 있습니다.
- `/ultrareview` 개선됨: 병렬화된 검사로 더 빠른 시작, launch dialog의 diffstat, 그리고 animated launching state를 제공합니다.
- mid-stream에서 stall되는 Subagent는 이제 조용히 hanging되는 대신 10분 후 명확한 오류로 실패합니다.
- Bash tool: 첫 번째 줄이 comment인 multi-line command는 이제 transcript에 전체 command를 표시하며, UI-spoofing vector를 종료합니다.
- `cd <current-directory> && git …`를 실행해도 `cd`가 no-op일 때 더 이상 permission prompt를 트리거하지 않습니다.
- 보안: macOS에서 `/private/{etc,var,tmp,home}` 경로는 이제 `Bash(rm:*)` allow rules 아래에서 위험한 removal 대상으로 취급됩니다.
- 보안: Bash deny rules는 이제 `env`/`sudo`/`watch`/`ionice`/`setsid` 및 유사한 exec wrapper로 감싼 command와 일치합니다.
- 보안: `Bash(find:*)` allow rules는 이제 `find -exec`/`-delete`를 자동 승인하지 않습니다.
- MCP concurrent-call timeout handling을 수정했습니다. 한 tool call의 메시지가 다른 call의 watchdog을 조용히 disarm할 수 있는 문제를 해결했습니다.
- Cmd-backspace / `Ctrl+U`를 수정하여 이제 cursor에서 line의 시작까지 다시 삭제하도록 했습니다.
- markdown table이 pipe character가 포함된 inline code span을 포함할 때 breaking되는 문제를 수정했습니다.
- unsent text를 composing하는 중에 session recap auto-firing되는 문제를 수정했습니다.
- `/copy` "Full response"가 GitHub, Notion 또는 Slack에 붙여넣기 위해 markdown table column을 align하지 않는 문제를 수정했습니다.
- running subagent를 보는 중에 입력된 메시지가 해당 transcript에서 숨겨지고 parent AI에 잘못 귀속되는 문제를 수정했습니다.
- Bash `dangerouslyDisableSandbox`가 permission prompt 없이 sandbox 외부에서 command를 실행하는 문제를 수정했습니다.
- `/effort auto` confirmation을 수정했습니다. 이제 "Effort level set to max"라고 표시되어 status bar label과 일치합니다.
- "copied N chars" toast가 emoji 및 기타 multi-code-unit character를 overcounting하는 문제를 수정했습니다.
- `/insights`가 Windows에서 `EBUSY`로 crashing되는 문제를 수정했습니다.
- exit confirmation dialog가 one-shot scheduled task를 recurring으로 잘못 레이블링하는 문제를 수정했습니다. 이제 countdown을 표시합니다.
- slash/@ completion menu가 fullscreen 모드에서 prompt border에 flush되지 않는 문제를 수정했습니다.
- `CLAUDE_CODE_EXTRA_BODY` `output_config.effort`가 effort를 지원하지 않는 모델에 대한 subagent call 및 Vertex AI에서 400 오류를 발생시키는 문제를 수정했습니다.
- prompt cursor가 `NO_COLOR`를 설정하면 disappearing되는 문제를 수정했습니다.
- `ToolSearch` ranking을 수정하여 pasted MCP tool name이 description-matching siblings 대신 실제 tool을 surface하도록 했습니다.
- compacting a resumed long-context session이 "Extra usage is required for long context requests"로 실패하는 문제를 수정했습니다.
- `plugin install`이 dependency version이 already-installed plugin과 충돌할 때 성공하는 문제를 수정했습니다. 이제 `range-conflict`를 보고합니다.
- "Refine with Ultraplan"이 transcript에서 remote session URL을 표시하지 않는 문제를 수정했습니다.
- SDK image content block이 process에 실패하면 session을 crash시키는 문제를 수정했습니다. 이제 text placeholder로 degrade됩니다.
- Remote Control session이 subagent transcript을 stream하지 않는 문제를 수정했습니다.
- Remote Control session이 Claude Code가 exit할 때 archived되지 않는 문제를 수정했습니다.
- Bedrock Application Inference Profile ARN을 통해 Opus 4.7을 사용할 때 `thinking.type.enabled is not supported` 400 오류가 발생하는 문제를 수정했습니다.
