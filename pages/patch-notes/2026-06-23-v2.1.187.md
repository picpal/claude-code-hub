> 원문: [Claude Code v2.1.187 Release Notes](https://github.com/anthropics/claude-code/releases/tag/v2.1.187)

## 변경사항

- sandboxed 명령이 credential 파일 및 secret 환경 변수를 읽는 것을 차단하는 `sandbox.credentials` 설정이 추가되었습니다
- model picker, `--model`, `/model`, 및 `ANTHROPIC_MODEL`에 조직에서 설정한 model 제한 사항이 추가되었으며, 제한된 model을 선택하면 "restricted by your organization's settings" 메시지가 표시됩니다
- fullscreen 모드에서 select 메뉴(permission 프롬프트, `/model`, `/config` 등)에 마우스 클릭 지원이 추가되었습니다
- 원래 `-p` 실행에서 model turn이 생성되지 않았을 때 `--resume`이 "No conversation found" 오류로 실패하는 문제가 수정되었습니다
- `--json-schema` 및 workflow `agent({schema})` structured output이 수정되었습니다. 이제 model이 성공적인 호출 후 `StructuredOutput`을 무한정 다시 호출할 수 없으며, follow-up turn은 이제 안정적으로 structured output을 반환합니다
- 5분 동안 응답이 없어 hang되는 remote MCP tool 호출이 수정되었습니다. 이제 무한정 차단되지 않고 대신 오류로 중단됩니다(`CLAUDE_CODE_MCP_TOOL_IDLE_TIMEOUT`으로 override 가능)
- agent proxy CA system-trust install이 추가된 후 Claude Code Remote 세션이 시작하는 데 약 2.7초 더 오래 걸리는 문제가 수정되었습니다
- 터미널이 paste를 per-byte extended-key 이벤트로 전달할 때 한국어/CJK 텍스트가 mojibake로 변환되는 문제가 수정되었습니다
- Remote Control을 통한 `/update`가 startup trust 대화가 표시되었을 때 hang되는 문제가 수정되었습니다
- agent가 structured output을 생성하지 않고 turn을 끝냈을 때 agents view의 background job이 "working" 상태에서 무한정 stuck되는 문제가 수정되었습니다
- agents view로 이동했다가 돌아올 때, 그리고 `/bg`, `/tui`, 또는 `/update` 후에 channel 연결이 끊어지는 문제가 수정되었습니다
- agent stop 알림이 agent를 중지한 사람을 올바르게 표시하지 못하는 문제가 수정되었으며, 표현이 개선되었습니다("came to rest" 대신 "finished"/"stopped")
- subagent depth 추적이 수정되었습니다. 이제 resumed subagent는 원래 spawn depth를 복원하고, forked subagent는 depth cap에 포함됩니다
- killed agent로부터 leaked agent worktree 등록이 수정되었습니다. locked `.git/worktrees/` 항목이 이제 자동으로 정리됩니다
- macOS의 Ghostty에서 fullscreen 모드에서 Cmd+click이 URL을 열지 않는 문제가 수정되었습니다
- `claude --help`이 `--bg`/`--background` flag를 나열하지 않는 문제가 수정되었습니다
- `/share`이 업로드 중일 때 Esc, Ctrl-C, 및 Ctrl-D가 작동하지 않는 문제가 수정되었습니다
- `/install-github-app`이 개선되었습니다. 이제 GitHub Actions workflow 설정은 선택 사항이므로 GitHub App만 설치하고 workflow/secret 단계를 건너뛸 수 있습니다
- `/btw`가 개선되었습니다. ←/→ 화살표 네비게이션을 사용하여 이전 답변을 단계별로 탐색할 수 있습니다
- `/plugin`이 개선되었습니다. 최근에 사용하지 않은 plugin을 표시하여 정리할 수 있습니다
- [VSCode] 큰 세션을 resume할 때 extension이 응답하지 않는 문제가 수정되었습니다
