## 변경사항

- `fallbackModel` 설정을 추가하여 primary model이 과부하되거나 사용 불가능할 때 시도할 최대 3개의 fallback model을 순서대로 구성할 수 있도록 함; `--fallback-model`이 이제 interactive session에도 적용됨
- deny rule tool-name 위치에서 glob pattern 지원 추가 (`"*"`는 모든 tool을 거부); allow rule은 non-MCP glob을 거부하며, deny rule의 알 수 없는 tool name은 시작 시 경고 표시
- cross-session messaging을 강화함: 다른 Claude session에서 `SendMessage`를 통해 relay된 메시지는 더 이상 user authority를 가지지 않음 — receiver는 relay된 permission request를 거부하며, auto mode는 이를 차단함
- `MAX_THINKING_TOKENS=0`, `--thinking disabled`, 그리고 model별 thinking toggle이 이제 Claude API를 통해 기본적으로 생각하는 model에서 thinking을 비활성화함 (3P provider는 변경 없음)
- Claude Code가 API가 예기치 않은 non-retryable error를 거부할 때 fallback model에서 한 번 재시도함; auth, rate-limit, request-size, transport error는 여전히 즉시 표시됨
- `claude update`는 이제 다운로드 전에 target version을 알리며 침묵하지 않음
- `claude agents`: list에 URL을 입력하면 이제 첫 번째 prompt가 해당 URL을 포함한 session으로 필터링됨
- 처리 불가능한 이미지가 session에 전송될 때 발생하는 반복적인 "image could not be processed" error 및 추가 token 사용량 수정
- startup 시 worker registration 중 brief backend 중단이 발생할 때 remote session이 영구적으로 stuck 상태가 되는 문제 수정
- 2026.1+에서 synchronized output을 활성화하여 JetBrains IDE terminal (IntelliJ, PyCharm, WebStorm 등)의 flickering 수정
- Kitty keyboard protocol (WezTerm, Ghostty, kitty)을 사용하는 terminal에서 Shift+non-ASCII 문자 (예: Shift+ä → Ä)가 손실되는 문제 수정
- Windows에서 killed process의 자식이 output pipe를 유지할 때 PowerShell command validation이 시간 초과를 훨씬 오래 지나서 hanging 되는 문제 수정
- macOS에서 daemon이 연결된 상태에서 죽을 때 orphaned `claude --bg-pty-host` process가 100% CPU에서 spinning하는 문제 수정
- voice mode에서 `/voice` toggle 후 stale auth check를 삭제하기 위해 `/login`이 필요한 문제 수정
- invalid entry를 가진 managed setting이 silent하게 남은 valid policy의 enforcement를 비활성화하는 문제 수정
- managed-settings `allowedMcpServers`/`deniedMcpServers` predicate이 `${VAR}` reference를 사용할 때 매칭되지 않는 문제 수정
- git worktree에 진입한 background agent session이 `claude agents`에서 다시 열릴 때 "No conversation found"로 crash-looping되는 문제 수정
- streaming 중 Ctrl+O transcript view에서 thinking text가 중복되는 문제 수정
- remote session 내에서 실행할 때 `/doctor`가 모순적인 failed "Not inside a remote session" check를 표시하는 문제 수정
- `claude agents` dispatch 및 reply input에서 multiline prompt를 입력할 때 cursor가 첫 번째 줄 끝에 stick되는 문제 수정
- Unicode를 지원하지 않는 terminal에서 background agent row 사이에 blank line이 나타나는 문제 수정
