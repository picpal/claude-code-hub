## 변경사항

- 스트림 중간 연결 끊김 수정: 부분 응답이 이제 보존되며 원본 오류를 표시하지 않으며, spinner가 "running tool"에서 더 이상 멈추지 않습니다
- WSL2의 Windows Terminal 및 VS Code에서 마우스 휠 스크롤 수정 (2.1.172의 회귀)
- sandbox `denyRead`/`allowRead` glob이 대규모 디렉터리 트리에 대해 Bash tool description을 엄청나게 크게 만들고 Linux에서 세션을 사용할 수 없게 만드는 문제 수정
- 피드백 설문조사가 turn 완료 직후 한 자리 숫자 응답을 세션 평가로 즉시 캡처하는 문제 수정
- 환영 화면에 여러 개의 홍보 배너가 쌓이는 문제 수정 — 이제 세션당 최대 하나의 프로모만 표시됩니다
- Ctrl+O가 subagent를 볼 때 subagent의 transcript를 표시하지 않는 문제 수정
- 프롬프트 입력을 클릭해도 subagent/footer 패널에서 포커스가 반환되지 않는 문제 수정
- 원격 세션 백그라운드 작업이 turn 사이에서 "still running"으로 멈춘 것처럼 보이는 문제 수정
- 원격 세션의 plugin 로딩 성능 개선
