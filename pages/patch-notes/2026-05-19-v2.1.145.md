## 변경사항

- `claude agents --json`을 추가하여 live Claude sessions을 JSON으로 나열할 수 있게 함 (tmux-resurrect, status bars, session pickers용)
- `claude_code.tool` OTEL spans에 `agent_id` 및 `parent_agent_id` 속성을 추가했으며, background subagent spans이 dispatching Agent tool span 아래에 중첩되도록 trace parenting을 수정함
- Status line JSON input에 GitHub repo 및 PR 정보가 감지될 때 포함되도록 함
- `/plugin` Discover 및 Browse 화면이 설치 전에 plugin의 commands, agents, skills, hooks, MCP/LSP servers를 표시하도록 함
- `claude agents` terminal tab 제목이 awaiting-input 개수를 표시하도록 함 (alt-tabbed window에서 agent가 입력을 기다리는지 알 수 있음)
- Slash command 및 @-mention suggestion list가 fullscreen mode에서 mouse hover 및 click을 지원하도록 함
- Stop 및 SubagentStop hook input에 `background_tasks` 및 `session_crons` 필드를 포함하도록 함
- Bash commands에서 allowlisted 되지 않은 환경 변수에 대한 bare variable assignments이 자동으로 승인되던 permission-prompt bypass를 수정함
- MCP prompt slash commands가 필수 인수가 생략될 때 raw server validation errors를 표시하던 문제를 수정함 — 이제 오류가 누락된 인수의 이름을 표시하고 예상 사용법을 보여줌
- Terminal이 resized되거나 refocused된 후 spinner 및 elapsed-time display가 keypress까지 freeze되던 문제를 수정함
- Default Windows PowerShell 5.1에서 cross-project resume hint가 실패하던 문제를 수정함 — Windows는 이제 `;`을 command separator로 사용함
- Agent view의 reply pane에서 voice push-to-talk가 작동하지 않던 문제를 수정함
- 여러 tasks가 한 번에 생성될 때 task lists가 random order로 rendering되던 문제를 수정함
- Marketplace가 이미 설치되었는데도 stale "Failed to install Anthropic marketplace" banner가 표시되던 문제를 수정함
- `gh pr create` 및 기타 PR-state-changing commands가 in-session에서 실행된 후 footer의 PR badge가 즉시 업데이트되지 않던 문제를 수정함
- Non-ASCII 이름을 가진 Agent Teams teammates이 invalid header encoding으로 인해 모든 API call에 실패하던 문제를 수정함
- `/review`가 Classic Projects가 있는 repos에서 오류가 발생하던 deprecated `projectCards` GraphQL query를 사용하던 문제를 수정함
- `claude plugin validate`가 파일을 가리키는 `skills:` entries를 flagging하지 않던 문제를 수정함 — 이제 parent directory를 제안함
- `context: fork`를 사용하는 skill이 실행되는 대신 반복적으로 자신을 re-invoke할 수 있던 infinite loop를 수정함
- Read tool을 개선하여 whole-file read가 token limit을 초과할 때 hard error 대신 "PARTIAL view" notice와 함께 truncated first page를 반환하도록 함
