> 원문: [Claude Code v2.1.136 Release Notes](https://github.com/anthropics/claude-code/releases/tag/v2.1.136)

## 변경사항

- 세션 품질 설문조사를 OpenTelemetry를 통해 응답을 캡처하는 엔터프라이즈를 위해 다시 활성화하도록 `CLAUDE_CODE_ENABLE_FEEDBACK_SURVEY_FOR_OTEL`이 추가되었습니다
- 사용자 의도나 허용 예외와 관계없이 무조건 차단하는 auto mode classifier 규칙을 위해 `settings.autoMode.hard_deny`가 추가되었습니다
- VS Code extension, JetBrains plugin, Agent SDK에서 `/clear` 후에 `.mcp.json`, plugin, claude.ai connector에 구성된 MCP server가 자동으로 사라지는 문제가 수정되었습니다
- 동시 credential 쓰기가 새로 회전된 OAuth token을 덮어쓰고 강제 재로그인을 유발할 수 있는 드문 로그인 루프가 수정되었습니다
- 여러 server가 동시에 refresh될 때 MCP OAuth refresh token이 손실되는 문제가 수정되었습니다 — 여러 원격 MCP server를 보유한 사용자는 더 이상 매일 재인증이 필요하지 않습니다
- tool call 후에 redacted thinking block을 내보낼 때 발생하는 API 오류(400)가 수정되었습니다
- 프로젝트 경로에 underscore가 포함되어 있을 때 `--resume` / `--continue`가 session을 찾지 못하는 문제가 수정되었습니다
- 일치하는 `Edit(...)` allow rule이 존재할 때 plan mode가 파일 쓰기를 차단하지 않는 문제가 수정되었습니다
- WSL2: xclip/wl-paste가 image 데이터를 읽을 수 없을 때 PowerShell fallback을 통해 Windows clipboard의 image paste가 이제 작동합니다
- cache cleanup이 실행 중인 session에서 여전히 사용 중인 version을 삭제할 때 plugin `Stop`/`UserPromptSubmit` hook이 실패하는 문제가 수정되었습니다
- slash command 대화의 시각적 일관성이 개선되었습니다: footer hint, dialog spacing, arrow-key styling이 표준화되었으며, dialog frame이 로딩 중에 팝인되지 않고 즉시 나타나도록 개선되었습니다
- bash command output 및 markdown code block에서 색상이 잘못된 위치에 나타나는 문제가 수정되었습니다
- ReasonML diff에서 word-diff boundary에 손상된 "undefined" text artifact가 렌더링되는 문제가 수정되었습니다
- worktree 제거 후 worktree exit 대화가 잘못된 디렉토리의 커밋되지 않은 파일에 대해 경고하는 문제가 수정되었습니다
- `@` file picker가 작은 non-git 디렉토리에서 mid-session에 생성된 파일과 일치하지 않는 문제가 수정되었습니다
- `@`-mention file picker가 100개 이상의 항목이 있는 디렉토리에서 파일을 찾지 못하는 문제가 수정되었습니다
- 실패한 tool call이 출력이 잘렸을 때 fullscreen mode에서 click-to-expand되지 않는 문제가 수정되었습니다
- persistent extended-key mode를 사용하는 terminal에서 Ctrl+G를 사용하여 외부 editor를 열 때 Backspace와 Ctrl+Backspace가 교환되는 문제가 수정되었습니다
- `/usage` 주간 reset이 calendar date 대신 시간을 표시하는 문제가 수정되었습니다
- welcome banner ellipsis가 CJK terminal에서 column overflow를 유발하는 문제가 수정되었습니다
- session history에 malformed input field를 포함한 tool call이 있을 때 `/insights` crash가 수정되었습니다
- tool의 collapsibility classification이 mid-session에 변경될 때 renderer crash가 수정되었습니다
- `plugin.json`의 `skills` entry가 plugin의 기본 `skills/` 디렉토리를 숨기는 문제가 수정되었으며, 파일 경로를 나열하면 자동으로 실패하는 대신 오류를 표시합니다
- IDE shell-integration lock file이 `CLAUDE_CONFIG_DIR`을 인식하지 않는 문제가 수정되었습니다
- 스트리밍 중 복사된 terminal output의 trailing whitespace가 수정되었습니다
- plugin uninstall 및 enable/disable이 slug를 case-insensitively 일치시키지 않는 문제가 수정되었습니다
- tool error truncation marker가 surrogate-pair string에 대해 음수 개수를 표시하는 문제가 수정되었습니다
- `CLAUDE_ENV_FILE` SessionStart hook의 env var가 `/resume` 또는 `/clear` 후에 stale되는 문제가 수정되었습니다
- `/branch`가 pasted multi-line name을 제공받을 때 multi-line session title을 저장하는 문제가 수정되었습니다
- wrapped text의 두 번째 줄에 stray leading space가 나타나는 문제가 column boundary에서 수정되었습니다
- Esc가 `/install-github-app`, `/desktop`, `/resume`, `/web-setup`의 대화를 닫지 않는 문제가 수정되었습니다
- `/doctor` MCP schema error가 누락된 field를 명시하지 않고 source file path를 표시하지 않는 문제가 수정되었습니다
- Bash permission prompt가 사용자가 읽을 수 있는 설명 대신 internal parser diagnostic을 표시하는 문제가 수정되었습니다
- space를 포함한 plugin slash command(예: `/myplugin review`)가 namespaced form으로 resolve되지 않는 문제가 수정되었습니다
- `AskUserQuestion`이 multi-select answer를 array로 제공받을 때 버리는 문제가 수정되었습니다
- `/clear <name>`이 `/resume`을 위해 cleared session을 label하지 않는 문제가 수정되었습니다
- `CronList` output이 qualifier 및 scheduled prompt를 누락하는 문제가 수정되었습니다
- "Jump to bottom" overlay가 fullscreen mode에서 CJK 문자에 color artifact를 남기는 문제가 수정되었습니다
- 넓은 markdown table이 스트리밍 중 terminal scrollback에 stale bordered render를 남기는 문제가 수정되었습니다
- pasted text placeholder를 포함한 긴 prompt가 auto-truncated될 때 pasted text가 자동으로 버려지는 문제가 수정되었습니다
- `/release-notes`가 실패한 changelog refresh 후에 이전 version에서 stuck되는 문제가 수정되었습니다
- `/mcp` server list가 terminal에 맞는 server보다 많을 때 스크롤되지 않는 문제가 수정되었습니다
- 초기 slash command 후에 mid-input slash command autocomplete가 작동하지 않는 문제가 수정되었습니다
- `autoScrollEnabled: false`를 사용하여 bottom으로 스크롤할 때 auto-follow가 다시 engage되는 문제가 수정되었습니다
- prompt suggestion이 Tab이나 arrow로 accept하지 않고 empty input에서 Enter로 auto-submit되는 문제가 수정되었습니다
- keyboard shortcut hint가 `keybindings.json`의 rebound key를 반영하지 않는 문제가 수정되었습니다
- `/settings` language change가 confirm 후 Escape에서 revert되는 문제가 수정되었습니다
- `/terminal-setup`이 exact name match 대신 partial prefix에서만 autocomplete에 나타나는 문제가 수정되었습니다
- `AskUserQuestion` 대화의 "Chat about this"가 question text를 지우는 문제가 수정되었습니다
- MCP tool result가 server가 content block을 반환할 때 invisible되는 문제가 수정되었습니다
- `--worktree`가 기존 또는 stale worktree와 충돌할 때 오류 메시지가 개선되었습니다
- plugin marketplace removal key가 retry와 충돌한 `r` 대신 delete와 일치하는 `d`로 변경되었습니다
