## 변경된 사항
- Anthropic Console(API 요금 결제) 인증을 위해 `claude auth login`에 `--console` 플래그를 추가했습니다
- `/config` 메뉴에 “턴 시간 표시” 토글 기능을 추가했습니다
- 명시적인 표준 입력(stdin)을 지정하지 않고 서브프로세스로 실행할 때(예: Python의 `subprocess.run`) `claude -p`가 응답하지 않는 문제를 수정했습니다.
- `-p`(인쇄) 모드에서 Ctrl+C가 작동하지 않던 문제를 수정했습니다
- 스트리밍 중 `/btw` 명령어가 실행될 때 부수적인 질문에 답변하는 대신 메인 에이전트의 출력을 반환하던 문제를 수정했습니다.
- `voiceEnabled: true`가 설정된 경우, 시작 시 음성 모드가 제대로 활성화되지 않던 문제를 수정했습니다
- `/permissions` 페이지의 좌우 화살표 탭 탐색 기능을 수정했습니다.
- `CLAUDE_CODE_DISABLE_TERMINAL_TITLE`이 시작 시 터미널 제목이 설정되는 것을 막지 못하던 문제를 수정했습니다.
- 워크스페이스 신뢰 설정으로 인해 사용자 지정 상태 표시줄이 차단될 때 아무것도 표시되지 않던 문제를 수정했습니다.
- 기업 사용자가 속도 제한(429) 오류 발생 시 재시도할 수 없던 문제를 수정했습니다.
- 대화형 `/resume` 명령어를 사용하여 세션을 전환할 때 `SessionEnd` 훅이 실행되지 않던 문제를 수정했습니다.
- 모든 시나리오에서 시작 시 메모리 사용량을 약 18MB 줄였습니다
- 시도당 2분의 타임아웃을 적용하여 비스트리밍 API의 대체 처리를 개선함으로써, 세션이 무한정 멈춰 있는 현상을 방지합니다
- `CLAUDE_CODE_PLUGIN_SEED_DIR`는 이제 플랫폼 경로 구분자(Unix에서는 `:`, Windows에서는 `;`)로 구분된 여러 시드 디렉터리를 지원합니다.
- [VSCode] `/remote-control` 명령어 추가 — 세션을 claude.ai/code에 연결하여 브라우저나 휴대폰에서 작업을 이어갈 수 있습니다
- [VSCode] 세션 탭에 이제 첫 번째 메시지를 기반으로 AI가 생성한 제목이 표시됩니다
- [VSCode] 응답이 완료된 후 ‘Thinking’ 알약에 “Thought for Ns” 대신 “Thinking”이 표시되던 문제를 수정했습니다.
- [VSCode] 왼쪽 사이드바에서 세션을 열 때 세션 비교(diff) 버튼이 표시되지 않던 문제를 수정했습니다.
