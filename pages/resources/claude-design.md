## 개요

Claude Design은 **텍스트 프롬프트로 시각 자산을 생성하는 독립 제품**입니다. 슬라이드 덱, 앱 프로토타입, 마케팅 원페이저 등을 대화형으로 만들 수 있으며, 음성·비디오·셰이더·3D·내장 AI를 지원하는 코드 기반 프로토타입도 제작 가능합니다.

Claude Code와는 별도 제품이지만, 디자인 → 코드 핸드오프를 통해 하나의 파이프라인으로 연결됩니다.

## 핵심 기능

### 입력 방식
- **텍스트 프롬프트**: 자연어로 디자인 설명
- **이미지 업로드**: 스크린샷, 목업, 레퍼런스 이미지
- **문서 업로드**: DOCX, PPTX, XLSX 파일
- **코드베이스 연결**: 기존 프로젝트의 디자인 토큰 활용

### 생성 가능한 산출물
- 앱 프로토타입 (음성, 비디오, 3D, 셰이더 포함)
- 프레젠테이션 / 피치 덱
- 마케팅 원페이저
- 대시보드 목업
- 인터랙티브 와이어프레임

## 사용 워크플로우

```
프롬프트 입력 → 초기 디자인 생성 → 수정/조정 → 내보내기 or Claude Code 핸드오프
```

### Step 1: 프로젝트 시작

프롬프트로 원하는 디자인을 설명합니다.

```
"SaaS 대시보드 랜딩 페이지를 만들어줘.
히어로 섹션에 CTA 버튼, 기능 소개 3개 카드, 
가격 테이블, 푸터 포함."
```

### Step 2: 초기 디자인 확인

Claude가 프롬프트를 기반으로 초기 디자인을 생성합니다. 생성된 결과를 확인하고 수정 방향을 결정합니다.

### Step 3: 반복 수정

대화, 인라인 주석, 직접 편집, 커스텀 슬라이더 등 다양한 방법으로 디자인을 조정합니다.

### Step 4: 최종 산출물

내보내기(PDF, PPTX, HTML 등) 또는 Claude Code로 핸드오프합니다.

## 디자인 시스템 온보딩

첫 사용 시 Claude가 팀의 **디자인 시스템을 자동으로 구축**합니다.

- 코드베이스와 디자인 파일을 읽어 색상, 타이포그래피, 컴포넌트 추출
- 이후 모든 프로젝트에 팀 디자인 시스템이 자동 적용
- 브랜드 일관성 유지

---

## 편집 및 수정

디자인 수정은 네 가지 방식으로 가능합니다.

| 방식 | 설명 |
|------|------|
| **대화** | 채팅으로 수정 요청 ("히어로 배경을 그라데이션으로 바꿔줘") |
| **인라인 주석** | 특정 요소를 선택하고 코멘트 |
| **직접 편집** | 텍스트, 레이아웃 등을 직접 수정 |
| **커스텀 슬라이더** | Claude가 생성한 조정 노브로 간격, 색상, 레이아웃을 실시간 조정 |

슬라이더로 부분 조정 후 "이 변경을 전체 디자인에 적용해줘"라고 요청하면 일괄 반영됩니다.

## Claude Code 핸드오프

Claude Design의 핵심 차별점은 **디자인에서 프로덕션 코드로의 직접 전환**입니다.

```
Claude Design (탐색/프로토타입)
    ↓ 핸드오프 번들 패키징
Claude Code (프로덕션 코드 구현)
```

### 핸드오프 프로세스

1. Claude Design에서 디자인 완성
2. 핸드오프 번들 자동 패키징 (디자인 토큰, 컴포넌트 구조, 에셋 포함)
3. Claude Code에 단일 명령으로 전달
4. Claude Code가 프로덕션 코드로 변환

이를 통해 **탐색 → 프로토타입 → 프로덕션 코드**까지 Anthropic 생태계 내에서 폐쇄 루프를 형성합니다.

## FAQ

**Q: Claude Design과 Claude Code는 같은 제품인가요?**
A: 아닙니다. Claude Design은 비주얼 디자인 도구이고, Claude Code는 코딩 에이전트입니다. 핸드오프 기능으로 연결됩니다.

**Q: 무료 플랜에서 사용할 수 있나요?**
A: 현재 Research Preview 기간 동안 Pro, Max, Team, Enterprise 플랜에서만 사용 가능합니다.

**Q: Figma를 대체하나요?**
A: Claude Design은 빠른 프로토타이핑과 시각 자산 생성에 초점이 맞춰져 있습니다. 정밀한 픽셀 단위 디자인 작업은 여전히 전문 디자인 도구가 적합할 수 있습니다.

**Q: 팀원과 공유할 수 있나요?**
A: 내부 URL로 조직 내 공유가 가능하며, PDF/PPTX/HTML 등으로 내보내기할 수 있습니다.
