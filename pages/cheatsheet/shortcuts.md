## 기본 단축키

| 단축키 | 동작 |
|--------|------|
| `Enter` | 메시지 전송 |
| `Shift+Enter` | 줄바꿈 |
| `Ctrl+C` | 현재 작업 중단 |
| `Ctrl+D` | 세션 종료 |
| `Up/Down` | 히스토리 탐색 |

## 편집 단축키

| 단축키 | 동작 |
|--------|------|
| `Ctrl+A` | 줄 처음으로 |
| `Ctrl+E` | 줄 끝으로 |
| `Ctrl+U` | 줄 전체 삭제 |
| `Ctrl+W` | 이전 단어 삭제 |

## 도구 권한

도구 실행 프롬프트에서:

| 키 | 동작 |
|----|------|
| `y` | 승인 |
| `n` | 거부 |
| `a` | 항상 승인 (이 세션) |
