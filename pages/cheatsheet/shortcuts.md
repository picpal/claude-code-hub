## 기본 단축키

| 단축키 | 동작 |
|--------|------|
| `Enter` | 메시지 전송 |
| `Shift+Enter` | 줄바꿈 |
| `Ctrl+C` | 현재 작업 중단 |
| `Ctrl+D` | 세션 종료 |
| `Up/Down` | 히스토리 탐색 |
| `Shift+Tab` | Plan 모드 / Accept 모드 전환 |
| `Escape` | 현재 응답 중단 |
| `Escape × 2` | 입력창 내용 삭제 후 직전 입력 복원 |

## 편집 단축키

| 단축키 | 동작 |
|--------|------|
| `Ctrl+A` | 줄 처음으로 |
| `Ctrl+E` | 줄 끝으로 |
| `Ctrl+U` | 줄 전체 삭제 |
| `Ctrl+W` | 이전 단어 삭제 |

## 도구 권한

도구 실행 프롬프트에서:

| 키 | 동작 |
|----|------|
| `y` | 승인 |
| `n` | 거부 |
| `a` | 항상 승인 (이 세션) |

## 이미지 입력

터미널에 스크린샷이나 이미지 파일을 드래그 앤 드롭하면 Claude가 해당 이미지를 직접 인식합니다.

**활용 팁**

- **무엇인지 파악**: UI 스크린샷을 드래그하면 화면 구성·컴포넌트 구조를 분석해 줍니다.
- **어떻게 구현할지**: 디자인 시안이나 에러 화면을 붙여 넣고 "이 화면을 코드로 구현해줘" 또는 "이 에러를 해결해줘"라고 입력하면 맥락을 그대로 이해하고 답변합니다.
- 지원 형식: PNG, JPG, GIF, WebP

## iTerm2 패널 단축키

iTerm2에서 패널을 분할해 Claude Code 실행 터미널과 작업 터미널을 나란히 사용할 수 있습니다.

| 단축키 | 동작 |
|--------|------|
| `Cmd+D` | 현재 패널을 세로로 분할 |
| `Cmd+Shift+D` | 현재 패널을 가로로 분할 |
| `Cmd+[` | 이전 패널로 이동 |
| `Cmd+]` | 다음 패널로 이동 |

## Bash 명령 빠른 실행

입력창에서 `!` 접두사를 붙이면 Claude를 거치지 않고 셸 명령을 즉시 실행합니다.

```
!npm run build
!git status
!ls -la
```

**특징**

- Claude의 응답 없이 명령 결과만 출력됩니다.
- 빌드·테스트·Git 상태 확인 등 반복 작업을 빠르게 처리할 때 유용합니다.
